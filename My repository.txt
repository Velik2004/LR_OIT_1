1. Git - система управління версіями з розподіленою архітектурою
2. "Мастер" або "невідстежуваний файл"
3. Команда "git add" використовують для створення робочого каталога в індексі наступного коміту.
4. Для перевірки статусу файлів, їх відстеження.
5. При виконанні команді git commit дані, які дабавленні за допомогою команди git add, пересосяться в репозиторій і встановлює посилання на ці дані.
6.Виконуючи команду git init, Git встановлюється для даного репозиторію.
7. В коментарях потрібно писати усі зміни, які було виконані
8. Потрібно написати команду git log
